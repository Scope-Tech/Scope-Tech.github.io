Pistacher Introduction

Pistacher是由Scope-Tech公司于2020年6月推出的Pistacher面向对象程序设计语言和Pistacher平台的总称。由squid233和同事们共同研发，并在2020年正式推出。

主要特性

  • Pistacher语言是简单的：
    Pistacher语言的语法与Java语言很接近，使得大多数程序员很容易学习和使用。

发展历史

  • 2020.6.12，Pistacher语言诞生

Pistacher开发工具

Pistacher语言尽量保证系统内存在1G以上，其他工具如下所示：

  • Linux 系统、Mac OS 系统、Windows 95/98/2000/XP，WIN 7/8/10系统。
  • Notepad 编辑器或者其他编辑器。

安装好以上的工具后，我们就可以输出Pistacher的第一个程序"Hello World！"

class HelloWorld {
    static main() {
        println("Hello World")
    }
}
