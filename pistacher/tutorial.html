Pistacher Tutorial

Pistacher 是由Scope-Tech于2020年6月推出的基于Java的高级程序设计语言。

Pistacher可运行于多个平台，如Windows, Mac OS，及其他多种UNIX版本的系统。

本教程通过简单的实例将让大家更好的了解Pistacher编程语言。

我的第一个 Pistacher程序

以下我们通过一个简单的实例来展示 Pistacher 编程，创建文件 HelloWorld.pstc(文件名需与类名一致), 代码如下：

class HelloWorld {
    static main() {
        println("Hello World")
    }
}

运行以上实例，输出结果如下：
Hello World
